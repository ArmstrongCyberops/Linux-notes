## Basic commands

## pwd
- Print working directory (shows your current location)

## ls
- List files and directories

## cd
- Change directory
Example : cd /home

## mkdir
- Create a new directory
Example : mkdir test-folder

## rm
- Remove files or directories
Example : rm file.txt

-=-=-=-=-=-=-=-=-

## File navigation

## .
- Current directory

## ..
- Parent directory
